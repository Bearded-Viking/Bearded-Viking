## [ Web-Based Task Manager: ](https://github.com/Bearded-Viking/TaskFlow)

A simplistic, no-nonsense, Work Flow Manager you never knew you needed. Simple, functional and aesthetic. <br>
Built with:
- Django 4.0.3
- Python 3.10.4
- SQL Lite 3

- CSS
